# ``OnePassword/Client``

A client for retrieving credentials from 1Password.

## Overview

The 1Password client wraps the `op` command-line tool to retrieve credentials
from your 1Password vault. It can retrieve usernames, passwords, and one-time
passwords (TOTPs) from a specified item.
